# Conferences

This directory contains conference summaries and recaps. Each conference should be in its own folder with the format:

```
YYYY-MM-DD-conference-name/
  index.qmd
  [optional images or other assets]
```

## Example Conference Structure

```
conferences/
  2025-03-14-tech-conference-2025/
    index.qmd
    session-notes.md
    photos/
```

## Conference Post Template

Create `index.qmd` in your conference folder:

```markdown
---
title: "Conference Name 2025"
author: "Ashutosh Bajpai"
date: "2025-03-14"
categories: [Conferences, Technology]
description: "My recap of Conference Name 2025"
---

Your conference summary here...
```
