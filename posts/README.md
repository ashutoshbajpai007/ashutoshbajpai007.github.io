# Posts

This directory contains all blog posts. Each post should be in its own folder with the format:

```
YYYY-MM-DD-post-title-slug/
  index.qmd
  [optional images or other assets]
```

## Example Post Structure

```
posts/
  2025-03-14-my-first-post/
    index.qmd
    image1.png
    image2.jpg
```

## Post Template

Create `index.qmd` in your post folder:

```markdown
---
title: "Your Post Title"
author: "Ashutosh Bajpai"
date: "2025-03-14"
categories: [Technology, Programming]
description: "A brief description of your post"
image: "thumbnail.jpg"  # optional
---

Your content here...
```
